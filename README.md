# 🎵 Papa Niko Bellic Music App

A premium, Spotify-inspired music streaming application named after the GOAT. Featuring a full-stack architecture with React (Vite) and Koa.js.

## 🚀 Live Demo
**Check out the frontend UI live here:**
[https://sridhar27-tech.github.io/niko-music/](https://sridhar27-tech.github.io/niko-music/)

> [!NOTE]
> The live demo is a static frontend deployment. For full audio playback and playlist persistence, clone the repository and run the backend server locally.

## 📦 Project Packages
### Frontend
- **React (Vite):** Core UI library
- **Axios:** API communication
- **Lucide-React:** Premium iconography
- **Framer Motion:** Smooth micro-animations

### Backend
- **Koa.js:** High-performance web framework
- **@koa/router:** Fast and flexible routing
- **Koa CORS:** Cross-origin resource sharing
- **Koa Bodyparser:** Request parsing
- **Nodemon:** Development automation

## 🛠️ How to Run Locally
1. **Clone the repository:**
   ```bash
   git clone https://github.com/sridhar27-tech/niko-music.git
   cd niko-music
   ```
2. **Setup Server:**
   ```bash
   cd server
   npm install
   npm run dev
   ```
3. **Setup Client:**
   ```bash
   cd ../client
   npm install
   npm run dev
   ```
4. **Enjoy!** Open [http://localhost:5173](http://localhost:5173) in your browser.
